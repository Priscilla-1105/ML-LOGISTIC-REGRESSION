# Machine Learning - Logistic Regression

## Overview

This project demonstrates the implementation of **Logistic Regression**, a supervised machine learning algorithm used for **binary classification** tasks. The workflow includes data preprocessing, feature engineering, model training, prediction, and performance evaluation using industry-standard machine learning techniques.

## Features

* Data preprocessing and cleaning
* Handling missing values
* Feature encoding and scaling
* Train-test data splitting
* Logistic Regression model training
* Prediction on test data
* Model evaluation using Accuracy Score, Confusion Matrix, and Classification Report

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

## Workflow

1. Import the dataset.
2. Perform data preprocessing and cleaning.
3. Handle missing values and encode categorical features (if required).
4. Split the dataset into training and testing sets.
5. Scale the features using StandardScaler (if applicable).
6. Train the Logistic Regression model.
7. Predict the target values for the test dataset.
8. Evaluate the model using accuracy and classification metrics.

## Model

**Algorithm:** Logistic Regression

**Task:** Binary Classification

## Evaluation Metrics

* Accuracy Score
* Confusion Matrix
* Precision

* F1-Score
